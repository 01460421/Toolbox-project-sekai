# 🎹 世界計畫工具箱 | SEKAI Toolbox

為 Project SEKAI 玩家打造的精緻工具集，採用奢華低調的侘寂黑金設計風格。

## ✨ 功能

### 📊 綜合力計算
- **綜合力計算機** - 區域道具 + CR + MySEKAI 門/娃
- **查卡系統** - 搜尋卡片、專精突破、無框畫

### 🏠 MySEKAI
- **素材整理** - 門/娃/家具素材總覽
- **門管理** - 各團體之門等級與加成
- **娃娃管理** - 角色娃娃等級設定
- **家具收集** - 家具收集進度追蹤

### 📈 成長系統
- **羈絆計算** - 羈絆升級經驗與獎勵
- **CR 等級** - 角色等級規劃
- **升等速度** - 根據遊玩時間預估進度

### 🏆 衝榜系統
- **資源規劃** - 火罐、結晶、時間估算
- **分析系統** - 即時排名追蹤
- **邊線預測** - 活動結算邊線預測

## 🚀 快速開始

```bash
# 安裝依賴
npm install

# 開發模式
npm run dev

# 建構
npm run build

# 啟動
npm start
```

## 📦 部署到 Vercel

1. Fork 此專案
2. 在 Vercel 中匯入
3. 自動部署完成
